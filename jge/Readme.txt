CONTENTS
========

This package contains the jGE NetLogo v2.0 extension.


INSTRUCTIONS
============

The jGE NetLogo extension folder should be copied in the extensions folder of 
the NetLogo installation. For instructions on using jGE NetLogo in your model
see the jGE NetLogo Documentation.
